# 麻雀点数○×ゲーム 手順書

## 目的
点数計算の練習用に、手牌と条件から正しい点数を答えるクイズ形式の機能です。

## 事前準備
- アプリを起動（例: `npm run dev`）
- ブラウザでアプリを開く

## 起動方法
1. 画面でスコアクイズページ（`/score-quiz`）を開く。
2. 問題が自動で表示されることを確認する。

## 画面の見方
- 牌表示エリア: 手牌 + 和了牌 + 鳴き（あれば）を表示。
- 条件表示エリア: 親/子、ツモ/ロン、場風/自風などの条件。
- 入力エリア: 点数入力欄。
- 判定エリア: 正誤と正解点数が表示される。

## 手順
1. 牌表示エリアと条件表示エリアを確認する。
2. 入力エリアに点数を入力する。
   - ロン: 支払い点数を1つ入力。
   - ツモ: 親/子の支払い点数を分けて入力。
3. 「判定」ボタンを押す。
4. 結果が「○/×」と正解点数で表示される。
5. 「次の問題」ボタンで次の問題に進む。

## 入力のルール
- 入力は数値のみ。
- 0や空欄は無効。
- 点数は100点単位に丸めた値を入力。

## 例
- 子ロンで 3900 点の場合: 3900 を入力。
- 子ツモで 子: 1000 / 親: 2000 の場合:
  - 子支払い = 1000
  - 親支払い = 2000

## よくあるミス
- 親/子の区別を間違える。
- ツモで支払い点数の入力を逆にする。
- 丸め後の点数ではなく、丸め前の点数を入力する。

## トラブルシューティング
- 問題が表示されない: ページを再読み込み。
- 判定が動かない: 入力欄が空か、数値以外が入っていないか確認。

## 補足
- 出題内容はプリセット問題を想定。
- 将来的にドラ/赤ドラ/鳴き等の条件も含む拡張を予定。

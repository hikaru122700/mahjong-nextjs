# 麻雀点数計算機 (Next.js版)

麻雀の点数を計算するWebアプリケーションです。Next.js + TypeScriptで実装されています。

## 機能

- 牌の選択（萬子、筒子、索子、字牌）
- 手牌13枚 + 和了牌1枚の入力
- 和了形の自動判定
- 役の自動検出
- 翻数・符・点数の計算
- 各種条件設定（ツモ/ロン、場風、自風、リーチ、一発、門前）

## インストール

```bash
npm install
```

## 開発サーバーの起動

```bash
npm run dev
```

ブラウザで [http://localhost:3000](http://localhost:3000) を開きます。

## ビルド

```bash
npm run build
```

## 本番環境での実行

```bash
npm start
```

## 使い方

1. **牌を選択**: 画面上部の牌をクリックして手牌を選択します（最大13枚）
2. **和了牌を選択**: 13枚選択後、和了牌を1枚選択します
3. **条件設定**: 和了方法（ツモ/ロン）、場風、自風などを設定します
4. **計算**: 「点数を計算する」ボタンをクリック
5. **結果確認**: 翻数、符、点数、成立役が表示されます

## 対応している役

### 1翻役
- リーチ、一発
- 門前清自摸和
- 断么九（タンヤオ）
- 平和（簡易判定）
- 役牌（三元牌、場風、自風）
- 一盃口（イーペーコー）

### 2翻役
- 七対子
- 対々和
- 三暗刻
- 混老頭
- 小三元
- 混全帯么九（チャンタ）
- 一気通貫（イッキツウカン）
- 三色同順（サンシキドウジュン）

### 3翻役
- 混一色
- 純全帯么九（ジュンチャン）
- 二盃口（リャンペーコー）

### 6翻役
- 清一色

## 注意事項

- 符計算は簡易的な実装です
- 鳴きの処理には対応していません
